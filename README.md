# Safarilink Clone

Static flight booking site (clone-style demo).

## Pages
1. index.html — Home + flight search
2. results.html — Flight results
3. seats.html — Seat selection
4. passengers.html — Passenger details
5. payment.html — Airtel Money / Card / Bank
6. confirmation.html — Booking confirmed + WhatsApp send

## Data
- airports.json — 23 airports (incl. Johannesburg, Pretoria, Durban, Mauritius, Maputo)
- flights.json — 190 flights, USD fares

## Config
- WhatsApp / Airtel pay number: +254 786 781665
- Currency: USD

## Run
    python3 -m http.server 8080
    open http://localhost:8080
